# ProActive Physiotherapy

A modern, full-stack web application designed for a premium physiotherapy clinic. This project features a responsive design, interactive appointment booking, user authentication, and a persistent database backend.

### Frontend (User Experience)
- **Modern & Responsive Design**: Built with a "Blue" premium theme, glassmorphism effects, and custom animations (`fade-in-up`, `sweep`).
- **Interactive Booking Flow**: A multi-step wizard for booking appointments with real-time cost calculation and selection states.
- **Content Pages**: Fully designed Home, About, Services, FAQ (with search), and Blog pages.
- **Contact Form**: Functional inquiry form with validation.

### Backend (Functionality)
- **User Authentication**: Secure Login and Registration system using **NextAuth.js**.
- **Database Storage**: Self-contained **SQLite** database (using `better-sqlite3`) for storing users, appointments, and messages.
- **Protected Routes**: "My Bookings" page is restricted to authenticated users.
- **API Routes**: RESTful endpoints for handling auth, bookings, and contact submissions.

## ğŸ Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites
- Node.js 18+ installed.

### Installation

1.  **Clone the repository** (or navigate to the project directory):
    ```bash
    cd proactive
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Initialize the Database**:
    This script creates the local SQLite database file (`proactive.db`) and sets up the necessary tables (`users`, `appointments`, `contact_messages`).
    ```bash
    node scripts/init-db.js
    ```

4.  **Start the Development Server**:
    ```bash
    npm run dev
    ```

5.  **View the Application**:
    Open [http://localhost:3000](http://localhost:3000) in your browser.
